# My Little Browser Game

Basit ve eğlenceli bir HTML5 tabanlı browser oyunu.

## 📋 Hakkında

Pure HTML5 ve Canvas teknolojileri kullanarak geliştirilmiş interaktif bir browser oyunudur. Harici kütüphane veya framework'e ihtiyaç duymadan çalışır.

## 🎮 Özellikler

- **Gerçek Zamanlı Oynanış** - Sorunsuz ve duyarlı kontrol
- **Canvas Grafikleri** - Smooth animasyonlar ve görsel efektler
- **Dark Theme UI** - Modern ve göz alıcı arayüz tasarımı
- **Responsive Design** - Tüm ekran boyutlarında uyumlu
- **Pure JavaScript** - Harici bağımlılık olmadan

## 🚀 Nasıl Oynanır

### Başlangıç
1. `My Little Browser Game.html` dosyasını tarayıcıda açın
2. Oyun otomatik olarak yüklenecektir
3. Tam ekran modunda oynamanız önerilir

### Kontroller
- **Mouse/Touchpad** - Hareket ve navigasyon
- **Sol Tık** - Etkileşim ve aksiyonlar
- **Arayüz Butonları** - Menü ve seçenekler

## 💻 Teknik Bilgiler

### Gereksinimler
- Modern web tarayıcısı (Chrome, Firefox, Safari, Edge)
- JavaScript ES6+ desteği
- HTML5 Canvas API desteği

### Dosya Yapısı
- `My Little Browser Game.html` - Oyunun tüm kodu (self-contained)

## 🎨 Tasarım

- **Stil**: Dark Mode tema
- **Font**: Exo 2 (Ana yazı tipi), Cinzel Decorative (Başlıklar)
- **Animasyonlar**: CSS keyframes ve smooth transitions
- **Layout**: Flexbox tabanlı responsive tasarım

## 🔧 Geliştirme

Oyun vanilla JavaScript ile yazılmıştır ve tek bir HTML dosyasından oluşur. Tüm CSS ve JavaScript kodu dosya içinde yer almaktadır.
