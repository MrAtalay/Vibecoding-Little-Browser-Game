# Vibecoding - Browser Games Collection

Tarayıcıda oynanan aksiyon ve strateji oyunları koleksiyonu.

## 📋 Proje İçeriği

Bu repository, HTML5 ve Canvas teknolojileri kullanarak geliştirilmiş interaktif browser oyunlarını içermektedir.

### Oyunlar

1. **Dungeon Blitz Series** - Düngeon stilinde aksiyon oyunları
   - `DungeonBlitz-1.html` - İlk versiyon
   - `DungeonBlitz-2.html` - 2. versiyon
   - `DungeonBlitz-3.html` - 3. versiyon
   - `DungeonBlitz-4.html` - 4. versiyon
   - `DungeonBlitz-6.html` - 6. versiyon
   - `DungeonBlitz-7.html` - 7. versiyon (Ultimate Edition)

2. **My Little Browser Game** - Hafif browser oyunu
   - `My Little Browser Game.html`

3. **Backup-MOBA-1.html** - MOBA tarzı oyun

## 🎮 Özellikler

### Dungeon Blitz
- **Combat System** - Gerçek zamanlı savaş mekanikləri
- **Leveling System** - XP ve level yükseltme
- **Stats Management** - HP, Rage, XP tracking
- **Shop System** - Silah ve item satın alma
- **Enemy AI** - Dinamik düşman davranışları
- **Beautiful UI** - Modern ve göz alıcı arayüz tasarımı

### Teknik Özellikler
- Pure HTML5 & Canvas - Harici kütüphane olmadan
- Responsive Design - Tüm ekran boyutlarında uyumlu
- Dark Theme UI - Modern karanlık tema
- Smooth Animations - Akıcı animasyonlar
- Real-time Updates - Anlık oyun güncellemeleri

## 🚀 Nasıl Oynanır

### Başlangıç
1. Oyunlardan birini seçin
2. HTML dosyasını tarayıcıda açın
3. Oyunun tam ekran modunda çalışması önerilir

### Kontroller
- **Mouse/Touchpad** - Hareket ve hedefleme
- **Sol Tık** - Saldırı
- **Arayüz Butonları** - Shop ve item yönetimi

## 💻 Teknik Bilgiler

### Gereksinimler
- Modern web tarayıcısı (Chrome, Firefox, Safari, Edge)
- JavaScript desteği (ES6+)
- Canvas API desteği
